# Tic-Tac-Toe
A GUI-based Python script for the classic "Tic-Tac-Toe" game!
### Details:  
Python Version: `Python 2.7`  
GUI framework: `Tkinter`  
Algorithm: `minimax`  
Defaults: Player goes first with cross (X) while the computer plays with noughts (O)
